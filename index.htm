Sub-K is a 932-byte Win32 static web server compiled with MASM. It serves files, supports basic MIME types, and handles concurrent clients.
Crinkler keeps the showcase build under 1 KB. A regular MASM/LINK build is included too.
Exchange Ideas
Sub-K includes numbered source-code stages, so you can jump into the project at different sizes and feature levels.
Did you know? Parts of Sub-K were written with DTE, another fully working sub-1KB experiment.
